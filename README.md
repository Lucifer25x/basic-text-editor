<br/>
<img src="./screen.png" width=500/>

## Functions: 
* Open File
* Open Folder
* Edit File
* Save File
* Create New File
* Create New Folder
* Get location of files and folders
* Make Window title with file path
* Toggle Sidebar
* Change editor configuration, keybindings and etc. (with config file)

## Packages:
* Electron.js
* Node-fs
* Node-path
* Ace.js (brace)
